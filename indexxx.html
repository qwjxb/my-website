что такое workflo,для чего он нужен и как работает?

Workflow-это схема котрая показывает шаги для достижения цели. Workflow нужен для того чтобы упрощать сложные задачи и делать их более понятными. Workflow работает как последовательность шагов которые нужно выполнить для того чтобы достигнуть цели

что такое git rebase,для чего он нужен и как работает?

git rebase- используется для перемещения и обьеденения коммитов на новую базу. git rebase позволяет сохранить историю изменения более понятной.

что такое git fetch, для чего он нужен и как работает?

git fetch-загружает изменения из удаленного репозетория это позволяет вам посмотреть каие изменения были внесены другими с помощью git fetch вы можете обновить информацию о состоянии удаленных веток, что полезно для командной работы.
